## Motion 挑战

Android Studio 4.0 为我们提供了全新的工具：`MotionEditor`，得益于它，我们创建动画变得十分简单，

在此我们发起一个 Motion 挑战，欢迎小伙伴发挥自己的想象力，将原创作品与我们分享。

### 参与方式

- fork 该项目
- 创建自己的动画并录制显示 gif 图
- 联系我们，我们将您的动画与项目链接展示在此项目的 README 中
